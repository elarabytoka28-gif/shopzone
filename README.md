# ⬡ ShopZone — Angular E-Commerce Application

A full-featured e-commerce web application built with **Angular 18** and **JSON Server** as a mock REST backend.

---

## 🚀 Tech Stack

| Layer | Technology |
|-------|-----------|
| Framework | Angular 18 (Standalone Components) |
| Styling | SCSS with CSS Variables |
| Forms | Reactive Forms + Template-driven Forms |
| State Management | RxJS BehaviorSubject |
| Backend | JSON Server (Mock REST API) |
| Routing | Angular Router (Lazy Loading) |

---

## ✨ Features

- 🔐 Login & Register with form validation
- 🛍️ Product listing with search and category filter
- 📦 Product detail page with quantity selector
- 🛒 Shopping cart with real-time updates
- ✅ Place orders and view order history
- 👤 User profile page
- 🔒 Protected routes with Auth Guard
- 📱 Fully responsive design

---

## ⚙️ Getting Started

### 1. Clone the repository
```bash
git clone https://github.com/elarabytoka28-gif/shopzone.git
cd shopzone
```

### 2. Install dependencies
```bash
npm install
```

### 3. Run the app
```bash
npm run dev
```

- Angular app → http://localhost:4200
- JSON Server → http://localhost:3000
